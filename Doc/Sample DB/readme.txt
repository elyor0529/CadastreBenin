Для просмотра примера БД необходимо использовать MSSQLServer версии не ниже 2012, минимум редакции SQL Server Express Localdb
Для развертывания БД можно:
1. Присоединить готовые файлы БД - DB_MSSQLServer_2012.zip
2. Вручную выполнить скрипты из файла DDL_MSSQLServer.zip
3. Развернуть приложение уровня данных - Data-tier Applications.dacpac
